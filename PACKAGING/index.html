Additional software for Debian 11 (Bullseye)

Add this repository to your APT sources list:
echo 'deb https://nodiscc.gitlab.io/toolbox/ bullseye main' | sudo tee /etc/apt/sources.list.d/toolbox.list && sudo apt update

Sources: https://gitlab.com/nodiscc/toolbox/-/tree/master/PACKAGING
