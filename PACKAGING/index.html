Additional software for Debian 12 (Bookworm)

Add this repository to your APT sources list:
wget -O- https://nodiscc.gitlab.io/toolbox/gpg.key | gpg --dearmor > nodiscc-toolbox.gpg
cat nodiscc-toolbox.gpg | sudo tee -a /usr/share/keyrings/nodiscc-toolbox.gpg > /dev/null
echo 'deb [signed-by=/usr/share/keyrings/nodiscc-toolbox.gpg] https://nodiscc.gitlab.io/toolbox/ bookworm main' | sudo tee /etc/apt/sources.list.d/nodiscc-toolbox.list
sudo apt update
